# KAIKKI (Kaikille Suomessa)

A comprehensive, multi-page city guide aggregator designed to streamline daily navigation, dining, and retail discovery across Finland. Built using a modular, performant architecture, KAIKKI centralizes disparate lifestyle services into an intuitive, high-fidelity interface.

**Live Demo:** [https://kaikki-one.vercel.app/](https://kaikki-one.vercel.app/)

## Core Modules

- **WHERE (Eating & Groceries):** A curated aggregate for delivery networks, local restaurant ecosystems, and grocery discovery.
- **WHEN (Journey Planner):** An integrated layout mapping public transport routing systems, taxi dispatch frameworks, and car rentals.
- **WHAT (Clothes & Accessories):** A comprehensive retail finder highlighting shopping malls, modern fashion hubs, and local secondhand markets.

## Tech Stack

- **Logic & Control:** Vanilla JavaScript (ES6+)
- **Styling Architecture:** Modular SCSS / CSS3 (Responsive Grid/Flexbox Systems)
- **Structure:** Multi-page Semantic HTML5
- **Deployment Pipelines:** Continuous Integration via Vercel
